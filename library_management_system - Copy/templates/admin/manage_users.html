Management Information

Use this panel to manage user accounts in the library system.

  • View detailed information about each user by clicking the eye icon.
  • Grant or revoke administrator privileges using the user plus/minus button.
  • Administrators can manage books, users, and view all library statistics.
  • Regular users can only borrow books and manage their own accounts.
Note: You cannot change your own admin status for security reasons.
